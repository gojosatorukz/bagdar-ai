# 🦾 Bagdar.AI — Intelligent Fitness Coach

> **Bagdar** (каз. *бағдар* — ориентир, направление). 
> Bagdar.AI — это персональный ИИ-тренер, который не просто пишет программы тренировок, а строит ваш личный маршрут к идеальному телу и отслеживает каждый шаг. Разработано в рамках хакатона.

![Bagdar.AI Preview](https://via.placeholder.com/1000x500/0b0f1a/60a5fa?text=Bagdar.AI+-+Intelligent+Coach) ## 🚀 О проекте

Современные фитнес-приложения либо слишком сложные, либо предлагают шаблонные программы. **Bagdar.AI** решает эту проблему, объединяя мощь современных LLM (Qwen 3) с удобным трекингом прогресса. ИИ анализирует физические параметры пользователя и генерирует уникальный план, а встроенный смарт-парсер превращает текст нейросети в интерактивный дневник.

### 💎 Ключевые фишки (Features)
* 🧠 **Smart AI Generation:** Генерация индивидуальных тренировочных планов и БЖУ с учетом веса, роста, опыта и целей (на базе API Alem).
* 🛡️ **Бронебойный парсер (Smart Parse v8.0):** Уникальный алгоритм на JS, который безошибочно извлекает дни недели и упражнения из свободного текста нейросети, отсеивая "воду" и советы.
* 📈 **Интерактивная аналитика:** Красивые графики роста силовых показателей (Chart.js) в реальном времени.
* 🔒 **Freemium Бизнес-модель:** Продуманная система монетизации с имитацией биллинга и защищенными Premium-маршрутами.
* 🎨 **Premium UI/UX:** Темная тема (Dark Mode), Glassmorphism, отзывчивый дизайн на Tailwind CSS.

## 💼 Бизнес-модель (B2C SaaS)

Проект изначально спроектирован как коммерческий стартап с моделью **Freemium**:
1. **Basic (Free):** Пользователь получает wow-эффект от бесплатной генерации ИИ-плана тренировок.
2. **Premium ($4.99/мес):** Разблокирует интерактивный дневник, запись подходов и графики прогресса. 
*Себестоимость генерации 1 плана через API стремится к нулю, что обеспечивает высочайшую маржинальность (Gross Margin) при масштабировании.*

## 🛠 Технологический стек

* **Backend:** Go (Golang) — *выбран за высокую производительность, строгую типизацию и скорость работы.*
* **Database:** PostgreSQL (через NocoDB) — *надежное хранение пользователей, хэшей паролей (bcrypt) и логов тренировок.*
* **Frontend:** HTML5, Tailwind CSS, JavaScript (Vanilla).
* **Charts:** Chart.js для визуализации данных.
* **AI Engine:** Qwen 3 (через Alem API).

## ⚙️ Установка и запуск (Local Setup)

Для запуска проекта на вашем устройстве потребуется установленный **Go (1.20+)**.

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/yourusername/bagdar-ai.git](https://github.com/yourusername/bagdar-ai.git)
   cd bagdar-ai

```

2. **Загрузите зависимости:**
```bash
go mod tidy

```


*(Убедитесь, что у вас установлены пакеты `github.com/lib/pq`, `golang.org/x/crypto/bcrypt`, `github.com/google/uuid`)*
3. **Запуск сервера:**
```bash
go run main.go

```


4. Откройте браузер и перейдите по адресу: [http://localhost:8080](https://www.google.com/search?q=http://localhost:8080)
